## 详细介绍 (200-500字)

BDE Score™ is an AI-powered quantitative analysis platform that provides transparent, multi-factor scoring for 73 stocks across US, HK, and A-share markets.

**What makes it different:**

Unlike black-box AI models, BDE Score™ uses 5 transparent factors — Momentum, Mean Reversion, Volume, Volatility, and Trend — each independently weighted and explainable. Every stock gets a composite score from 0-100 with clear signal classification (Bullish/Neutral/Bearish).

**Key features:**
- 🌏 Three markets, one view: US (25 stocks), HK (26 stocks), A-shares (23 stocks)
- ⚡ Real-time dual-channel data: FutuOpenD primary + Sina Finance fallback
- 🔒 EU AI Act Art.50 compliant by design (full audit trails, explainable methodology)
- 🔌 Embeddable widgets: anyone can embed live scores on their website
- 📊 Shareable score cards: URL-based SVG cards for social sharing
- 💰 USDC payments on Base chain — no gatekeepers

**Pricing:**
- Free: Dashboard + 3 API queries/day
- Premium: $29/month (unlimited API + 365-day history)
- Institutional: $199/month (custom universe + compliance reports + SLA)

**Open source:** https://github.com/hbhqq9/bde-score

⚠️ Technical analysis tool only. Not financial advice.
