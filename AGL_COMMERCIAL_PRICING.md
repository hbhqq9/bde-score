## 服务分层

### Tier 0: Free — 合规自检（Lead Gen）
- **价格**: $0
- **内容**:
  - Art.50 Self-Checker（10题问卷，即时评分）
  - Compliance Whitepaper（完整条款解读）
  - 开源参考实现（BDE Score代码库）
- **目的**: 获客 → 识别不合规企业 → 转化为付费客户
- **渠道**: GitHub Pages + Dev.to + Hacker News

### Tier 1: Compliance Report — 合规审计报告
- **价格**: $199 / 次
- **内容**:
  - 针对客户AI系统的Art.50合规评估
  - 缺失项清单 + 修复建议
  - 合规差距评分（0-100）
  - 执行优先级排序
- **交付**: PDF报告 + 30分钟视频解读
- **周期**: 48小时内交付
- **成本**: ~$0.05（AI生成 + 人工审核）
- **毛利率**: >99%

### Tier 2: Implementation Package — 合规实施包
- **价格**: $999 / 次
- **内容**:
  - Tier 1全部内容
  - 代码级合规实现（HTTP headers + JSON embedding + audit log）
  - 合规模板集成（适配客户技术栈）
  - 3次30分钟技术对接会议
- **交付**: 代码PR + 文档 + 部署指南
- **周期**: 5个工作日
- **成本**: ~$5（AI生成 + 人工审核）
- **毛利率**: >99%

### Tier 3: Continuous Monitoring — 持续合规监控
- **价格**: $149 / 月
- **内容**:
  - 实时合规状态监控
  - 链上合规证明自动记录
  - 法规变更自动检测+告警
  - 月度合规报告
  - 季度合规审计
- **交付**: Dashboard + API + 邮件报告
- **周期**: 按月订阅
- **成本**: ~$2/月/客户（API调用 + 链上gas）
- **毛利率**: >98%
