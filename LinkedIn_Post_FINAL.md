**1. "It only applies to high-risk AI"**
Wrong. ALL AI systems generating synthetic content for natural persons are in scope. Chatbots, image generators, translation tools — all covered.

**2. "A disclaimer is enough"**
Wrong. The Code of Practice requires **multi-layer marking architecture**: C2PA metadata + invisible watermarks + fingerprinting. Single-technique = non-compliant.
