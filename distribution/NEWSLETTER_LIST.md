# Newsletter Submission Targets

## Python/Developer Newsletters

### Python Weekly
- Website: https://www.pythonweekly.com/
- Submit: Email to newsletter@pythonweekly.com or via website contact
- Frequency: Weekly
- Audience: 100K+ Python developers

### Import Python
- Website: https://importpython.com/newsletter/
- Submit: Via website submission form
- Frequency: Weekly
- Audience: Python developers

### Changelog
- Website: https://changelog.com/
- Submit: https://changelog.com/news/submit
- Audience: Developers, open source enthusiasts
- Very receptive to new open source projects

### TLDR
- Website: https://tldr.tech/
- Submit: sponsorship@tldr.tech or via website
- Multiple editions: Python, Web Dev, DevOps, AI
- Audience: 500K+ across all editions

### Hacker Newsletter
- Website: https://hackernewsletter.com/

### DevOps Weekly
- Website: https://www.devopsweekly.com/

- Audience: DevOps engineers

### Status Code
- Website: https://statuscode.tech/
- Submit: Via website
- Audience: .NET and general developers

## Submission Priority
1. **Changelog** - Most receptive to new OSS projects
2. **TLDR Python** - Large audience, relevant
3. **Quantocracy** - Perfect fit for trading/quant audience
4. **Python Weekly** - Large Python audience
5. **Import Python** - Secondary Python audience

## Submission Tips
- Lead with the problem solved, not features
- Mention it's MIT licensed and open source
- Include a one-line command to try it
- Reference EU AI Act compliance for credibility
- Keep emails short and scannable
