## Description
BDE Score is an open-source multi-factor stock analysis system that covers 74 stocks across three major markets: US (25), Hong Kong (26), and China A-shares (23). It provides comprehensive scoring across fundamentals, technicals, risk, and sentiment factors.

## Key Features
- 74 stocks across US, HK, and China A-share markets
- Multi-factor analysis: fundamentals, technicals, risk, sentiment
- Live API with zero registration required
- shields.io-compatible dynamic badges for README files
- GitHub Action for CI/CD integration
- EU AI Act Art.50 compliant transparency labels
- USDC payment on Base Chain (1 USDC = 1 year access, 2,592,000 API calls)
- API Key management (generate, revoke, list)
- Docker ready with SQLite persistent storage
