# Slant.co Submission - BDE Score

## Question: "What are the best open-source stock analysis tools?"

### BDE Score

**Pros:**
- ✅ Covers 3 markets (US/HK/A-shares) with 74 stocks
- ✅ Zero registration - just API calls
- ✅ EU AI Act Art.50 compliant
- ✅ shields.io badges for easy integration
- ✅ GitHub Action available
- ✅ Multi-factor scoring (fundamentals, technicals, risk, sentiment)
- ✅ Open source (MIT license)
- ✅ USDC payment option (decentralized)
- ✅ Self-hostable with Docker

**Cons:**
- ⚠️ SQLite backend (not suitable for very high throughput)
- ⚠️ Hong Kong data currently limited to FutuOpenD dependency
- ⚠️ Sentiment analysis uses basic keyword matching

**Best for:**
- Individual developers building trading tools
- Researchers needing multi-market data
- Teams wanting compliant AI transparency labels
- Open source enthusiasts
