# DevPost Project Listing

## Project Name
BDE Score™

## Short Description
AI-powered multi-market stock analysis with transparent multi-factor scoring. Covers 73 stocks across US, HK, and A-share markets. EU AI Act Art.50 compliant.

## Full Description
BDE Score™ is a quantitative stock scoring system that brings transparency to AI-powered stock analysis.

### What It Does
Evaluates 73 stocks across US, HK, and China A-share markets using 5 explainable factors:
- Momentum, Mean Reversion, Volume, Volatility, Trend
- Each stock gets a composite score (0-100) with signal classification
- Full factor breakdown visible for every score — no black boxes

### Built With
- Python (FastAPI)
- FutuOpenD + Sina Finance (dual data channels)
- Cloudflare Tunnel (infrastructure)
- Base chain / USDC (payments)
- bcrypt (security)

### Why It Matters
1. EU AI Act Art.50 compliance from day one (Aug 2, 2026 deadline)
2. First transparent stock scoring tool covering 3 major markets
3. Open source (MIT) — fully auditable
4. Crypto-native payments — no banking dependency
