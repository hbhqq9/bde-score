# BDE Score™ 积分计费系统

## 定价模型

### 免费层（Free）
- **额度**：3次/天，无需注册
- **覆盖**：73只股票基础评分
- **目的**：体验+转化漏斗入口
- **限制**：无历史数据、无API Key

### 积分层（Starter → Pro → Institutional）

| 档位 | 积分 | 价格(CNY) | 单价 | 折扣 | 分析次数 |
|------|------|----------|------|------|---------|
| 体验包 | 1,000 | ¥10 | ¥0.010/积分 | — | 100次 |
| 标准包 | 10,000 | ¥90 | ¥0.009/积分 | 9折 | 1,000次 |
| 专业包 | 100,000 | ¥800 | ¥0.008/积分 | 8折 | 10,000次 |
| 机构包 | 1,000,000 | ¥6,000 | ¥0.006/积分 | 6折 | 100,000次 |

### 消耗规则
- **每次分析调用** = 10积分（¥0.10 at 体验包价）
- **查询余额/流水** = 免费
- **健康检查/市场状态** = 免费

### 单位经济学
```
单次分析成本：~¥0.10（10积分 × ¥0.01/积分）
单次分析价值：传统研报 ¥500-5000
价值倍率：50x-500x
```

## 技术架构

```
Client → FastAPI → CreditManager.check() → FactorEngine → 行情数据
                ↓
         SQLite: user_credits + credit_ledger
                ↓
         Balance >= 10? → deduct → analyze → return
         Balance < 10?  → 402 Payment Required
```

## 支付接入路线图

### Phase 1: 手动充值（当前）
- 用户通过飞书/微信联系 → 确认付款 → 手动添加积分
- 适合早期种子用户，验证付费意愿

### Phase 2: 微信支付/支付宝
- 接入微信支付商户平台（需营业执照）
- 或使用聚合支付（如 PayJS / 虎皮椒）

### Phase 3: 自动化
- 充值 → 即时到账 → API调用 → 余额不足自动提醒
- 月度账单自动生成
