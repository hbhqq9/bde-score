Title: BDE Score™ – Open-source multi-factor stock scoring (US/HK/A-share, 74 stocks)
URL: https://github.com/hbhqq9/bde-score
Tags: python, finance, opensource, api

Description:
Real-time composite score (0-100) combining momentum, volatility, volume, trend, and risk factors. REST API with no signup required, shields.io badges, and a GitHub Action. Covers 25 US + 26 HK + 23 A-share stocks. Built with Python/FastAPI + FutuOpenD + SQLite. EU AI Act Art.50 compliant with explainable factor breakdowns.
