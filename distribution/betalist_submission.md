Product Name: BDE Score™
Tagline: Open-source multi-factor stock scoring API — US, HK & A-Share markets
Website: https://github.com/hbhqq9/bde-score
Description: BDE Score™ provides real-time composite stock scores (0-100) combining momentum, volatility, volume, trend and risk analysis. Covers 74 stocks across US, Hong Kong and China A-share markets. Free REST API with no signup, embeddable badges, and GitHub Action integration. Built for developers, quant researchers and fintech builders who need transparent, explainable stock analysis without $20K/year Bloomberg subscriptions.
