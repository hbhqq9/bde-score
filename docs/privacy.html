BDE Score™ Privacy Policy

Last updated: July 11, 2026

Overview

BDE Score™ is a stock analysis MCP (Model Context Protocol) server providing quantitative scoring for US, HK, and CN markets.

Data Collection

BDE Score™ does not collect, store, or transmit personal user data:

  • No user registration or authentication required
  • No IP addresses, user agents, or session data collected
  • No cookies or tracking technologies
  • No query history or analysis results stored
  • No data shared with third parties

Data Processing

  1. Your MCP client sends stock symbols or market filters
  2. Our server computes quantitative scores using publicly available data
  3. Results returned in real-time
  4. No data persisted after response delivery

Data Sources

All analysis based on publicly available financial data. Scores are derived from quantitative factors and do not constitute financial advice.

Third-Party Services

BDE Score™ does not integrate with or send data to any third-party services.

Contact

For privacy questions: GitHub Issues

Disclaimer

BDE Score™ is a technical service for quantitative stock analysis. It does not constitute financial advice. Always consult a licensed financial advisor before making investment decisions.
