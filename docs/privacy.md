# BDE Score™ Privacy Policy

**Last updated: July 11, 2026**

## Overview

BDE Score™ is a stock analysis MCP (Model Context Protocol) server that provides quantitative scoring and analysis for stocks across US, HK, and CN markets. This privacy policy explains how we handle data.

## Data Collection

BDE Score™ does **not** collect, store, or transmit any personal user data. Specifically:

- We do **not** require user registration or authentication
- We do **not** collect IP addresses, user agents, or session data
- We do **not** use cookies or tracking technologies
- We do **not** store query history or analysis results
- We do **not** share data with third parties

## Data Processing

When you use BDE Score™:

1. Your MCP client sends a request containing stock symbols or market filters
2. Our server computes quantitative scores using publicly available market data
3. Results are returned to your MCP client in real-time
4. No data is persisted after the response is delivered

## Data Sources

All analysis is based on publicly available financial data. BDE Score™ does not provide licensed financial data feeds. All scores are derived from quantitative factors and should not be interpreted as financial advice.

## Third-Party Services

BDE Score™ does not integrate with or send data to any third-party services. The MCP server operates as a standalone computational endpoint.

## Contact

For privacy-related questions, please open an issue on our [GitHub repository](https://github.com/hbhqq9/bde-score/issues).

## Disclaimer

BDE Score™ is a technical service for quantitative stock analysis. It does not constitute financial advice, investment recommendation, or brokerage service. Always consult a licensed financial advisor before making investment decisions.
