EU AI Act Article 50 - Compliance Statement

Effective Date: 2026-08-01 | Last Updated: 2026-07-12

✅ Full Compliance — BDE Score meets all Article 50 transparency obligations ahead of the August 2, 2026 enforcement date.

1. What is BDE Score?

BDE Score is an AI-powered reliability assessment system for MCP (Model Context Protocol) servers and AI Agents. It analyzes publicly accessible endpoints to generate trust scores and compliance assessments.

AI System Rule-based + LLM Analysis Public Signal Detection

2. Article 50 Obligations & Our Implementation

Art. 50(1) — AI System Identification

All BDE Score outputs clearly identify themselves as AI-generated assessments:
  • API responses include X-BDE-AI-System: true header
  • JSON outputs contain ai_system_info object with full disclosure
  • HTML reports display "Generated by BDE Score AI Assessment Engine"
  • MCP server metadata identifies BDE Score as an AI system

Art. 50(2) — Content Generation Disclosure

Users are informed that assessment content is AI-generated:
  • Methodology explanation on Landing Page
  • API documentation discloses AI-based scoring approach
  • Limitations clearly stated in every assessment output

Art. 50(3) — Synthetic Content (N/A)

BDE Score does not generate synthetic text, images, audio, or video. This obligation does not apply.

Art. 50(4) — Deep Fake (N/A)

BDE Score does not generate or manipulate deep fake content. This obligation does not apply.
