# BDE Score™ Badge API

## Embeddable Badges for Your README

### Market Overview Badge
```markdown
![BDE Score](https://atlantic-remains-atomic-floor.trycloudflare.com/api/badge?market=ALL)
```

### Individual Stock Badge
```markdown
![AAPL Score](https://atlantic-remains-atomic-floor.trycloudflare.com/api/badge?symbol=AAPL)
```

### How It Works
- Real-time BDE Score displayed as a shield badge
- Color-coded: Green (BUY) / Yellow (HOLD) / Red (SELL)
- Auto-updates every 15 minutes
- Free to use, no API key required

### Example Output
- 🟢 AAPL: 60.9 (BUY)
- 🟡 TSLA: 50.2 (HOLD)
- 🔴 LI: 40.2 (SELL)
